# Altheryx
TXT com as automações feitas em Altheryx


Este repositório contém programas desenvolvidos no Alteryx, exportados e versionados no formato .txt, com o objetivo de facilitar o controle de versão, auditoria de mudanças e colaboração em equipe.
📌 Sobre o Projeto
Os arquivos .txt representam workflows criados no Alteryx, permitindo:

Versionamento via Git/GitHub
Comparação de alterações entre versões
Armazenamento estruturado de lógicas de ETL, automações e análises


⚠️ Observação: Os arquivos .txt não são executáveis diretamente. Para execução e edição visual, é necessário importar o conteúdo no Alteryx Designer.

▶️ Como Utilizar

Abra o Alteryx Designer
Crie um novo workflow ou abra um existente
Importe ou substitua o conteúdo do workflow pelo arquivo .txt correspondente
Valide as conexões de dados, paths e credenciais antes da execução

✅ Pré-requisitos

Alteryx Designer (versão compatível com o workflow)
Permissões de acesso às fontes de dados utilizadas
Ajuste de caminhos locais ou variáveis de ambiente, quando aplicável

📝 Boas Práticas

Sempre documentar alterações relevantes nos commits
Evitar salvar credenciais sensíveis nos arquivos
Utilizar nomes de arquivos claros e padronizados
Manter comentários explicativos dentro do workflow quando possível

🧾 Versionamento
O versionamento segue o controle padrão do Git. Recomenda-se:

Commits pequenos e descritivos
Uso de branches para desenvolvimento e testes
Revisão antes de merge em branch principal
